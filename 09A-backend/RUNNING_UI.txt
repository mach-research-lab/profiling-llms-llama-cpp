Step 1:
Open two terminals

Step 2:
In one of the terminals, navigate to 09A-backend directory and run the command:
" source .venv/bin/activate "  (Starting virtual enviorment)
followed by:
" python3 server.py "

This will start the server on port 8000, make sure that it is forwarded

Step 3:
In the other terminal, navigate to 09A-frontend directory and enter the following command:
" python3 -m http.server 8080 "
This will serve the frontend to the port 8080, it should work with any port of your choice as long as it's forwarded.

Then to see the frontend enter : 
" localhost:8080 " (or port of you choice)

Worth noting is that the meassured data displayed in the UI is currently fake
